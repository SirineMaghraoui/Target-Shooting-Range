# Target-Shooting-Range
An FPS game that provides the player with a shooting range environment with targets to shoot at like a soldier. Along with shooting, the player can move, walk, run and jump around the place. The player can also switch between rifles, guns, grenades, and knife. They can reload the chosen weapon too if they are out of ammo. The bullets and grenades leave a damage trait on the target surface.
